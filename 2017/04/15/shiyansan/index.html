实验三 使用Github管理Android项目

一、实验目的

  1. 掌握Android Studio中Git使用
  2. 掌握Github管理项目

二、实验任务

  1. 在Android Studio中新建项目
  2. 添加VCS中的Git支持
  3. 发布项目到Github
  4. 在Android Studio中进行修改代码、提交、新建分支、合并等操作
  5. 推送项目到Github
  6. 如有多人合作，处理pull request

三、实验步骤与结果

  1. 在Android Studio中新建项目

  1. 添加VCS中的Git支持

​ （1）打开Android Studio.

​ （2）选择VCS下的import intoVersionControl，再选择ShareProjecton Github

​ （3）登录Github

​ （4）点击Share，将Android分享到Github上

（5）点击OK，将所有文件分享到Github上

​ （6）跳出Add File toGit对话框，选择Yes

​ （7）Android工程成功推送后，Android Studio的右上角会出现提示

​ （8）在github上，可看到已成功推送的工程

​ （9）在Android上修改代码，右键，选择GitàCommit file，输入Program change，点击commit and push.

四、实验小结

通过这次实验，我掌握了将Android代码推上Github的技能，这样方便将代码保存下来，只要有网，就不用害怕代码不再身边，也不用担心代码丢失。
